Flashcard 1

O que significa o termo "a11y" e qual é o seu objetivo principal?

Resposta

É o numerônimo para Accessibility (A + 11 letras + Y). O objetivo é garantir que a web seja utilizável por qualquer pessoa.

Flashcard 2

Por que toda imagem em HTML precisa do atributo alt, mesmo que vazio?

Resposta

  • Para leitores de tela de usuários cegos.
  • Como texto alternativo caso a imagem falhe.
  • Identificar imagens puramente decorativas (alt="").
